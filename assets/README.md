# Assets Folder

This folder contains custom images and icons for the BuyBuddy application.

## BuyBuddy Icon

To add a custom icon for BuyBuddy:
1. Place your icon image in this folder as `buybuddy.png`
2. Recommended size: 100x100 pixels or larger (square dimensions work best)
3. Supported formats: PNG (recommended for transparency), JPG, SVG

If no custom icon is provided, the application will display a shopping cart emoji (ðŸ›’) by default.
